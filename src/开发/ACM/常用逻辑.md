## 输入
```
a=input()//输入捕获
```
## 输出
```
print(".5f"%a)//输出特定小数位
print("%.5f" %a,"%.5f" %a,"%.5f" %a)//多位输出
```
## 数学运算
```
result = a / b # 普通除法
result = a // b # 整除
result = a % b # 取余
```
## 循环逻辑
```
fruits = ["apple", "banana", "cherry"]  
for fruit in fruits:  
    print(fruit)

for i in range(5):  # 从 0 到 4
    print(i)
    
for i in range(1,5):  # 从 1 到 4
    print(i)
```
